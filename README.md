This is a personal portfolio built with React (TypeScript) and Vite. It includes sections for experiences, education, projects, and a simple blog/posts list. Social links for LinkedIn, GitHub, and Kaggle are included.

## Features
- Responsive layout with a clean, modern design
- Experience and education timelines
- Project grid with short descriptions and stack
- Blog/posts section (static entries; can be wired to a CMS later)
- Social chips: LinkedIn, GitHub, Kaggle

## Getting Started
- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build for production: `npm run build`

## Customization
- Edit content in `src/App.tsx` arrays: `experiences`, `education`, `projects`, `posts`, and `socials`.
- Update styling in `src/App.css` and global styles in `src/index.css`.

## Deployment
- Any static host works (Netlify, Vercel, GitHub Pages). Publish the `dist/` directory generated by `npm run build`.
